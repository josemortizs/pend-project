Organiza tus notas y tareas pendientes en tu Panel de Administración WordPress.

== Descripción ==

Pend Project es un sencillo plugin para el control de tareas y notas en las que almacenar tareas pendientes de realizar
en base a un proyecto. Por ejemplo: Del proyecto www.ortizsanchezdev.es, añadir un logotipo al header, prioridad: Alta,
periodicidad: Puntual.

Las notas pueden moverse de sitio, por lo que podrás ir apilando las que vayas terminando o borrarlas directamente. 

Diseñé este plugin para quién trabajase muchas horas en el Panel de Administración de WordPress, para que pudiese tener 
esta utilidad sin usar una herramienta externa.

== Instalación ==


1. Sube la carpeta "pendproject" al directorio `/wp-content/plugins/`, de tu instalación WordPress, 
   o instálalo directamente desde el repositorio de Plugins WordPress.
2. Activa el plugin desde la opción de Plugin en el Panel de Administración WordPress
3. Dirígete a la nueva sección creada en el Panel de Administración, "Gestionar Pend Project".
4. ¡Crea tu primera tarjeta!


== Preguntas frecuentes ==

= ¿Si muevo las tarjetas por la pantalla mantendrán su nueva ubicación al volver a abrir pendproject? =

No, estudiaré la viabilidad de esta funcionalidad para una futura revisión del plugin.

= ¿Funciona correctamente en dispositivos móviles? =

Sí, de hecho es fundamental para la filosofía del plugin, recuerdas algo pendiente, abres tu panel de administración,
creas tu tarjeta, eliges periodicidad, prioridad y te olvidas, ya lo tienes registrado como pendiente.
